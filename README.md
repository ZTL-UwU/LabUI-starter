# LabUI starter

Starter template for [LabUI](https://github.com/ZTL-UwU/LabUI)

## Get Started

```
npm i
npm run dev
```

## Commands
| Command   | Description                                   |
| :-------- | :-------------------------------------------- |
| `dev`     | Development (port 8080)                       |
| `build`   | Build your app for production!                |
